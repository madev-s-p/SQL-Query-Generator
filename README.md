# SQL Query Generator – Natural Language to SQL

SQL Query Generator is an intelligent system that converts **natural language queries into SQL statements automatically**.  
This project helps users interact with databases using plain English instead of writing complex SQL queries manually.

The system uses **Natural Language Processing (NLP)** techniques along with rule-based logic to identify database tables, columns, and conditions, then generates the appropriate SQL query.

This project is designed as a **learning project for database querying and AI-powered interfaces** and demonstrates how conversational interfaces can simplify data access.
